# MERN Stack Developer Portfolio

A modern, eye-catching portfolio website for a professional software developer built with the MERN stack (MongoDB, Express.js, React, Node.js).

## 🚀 Features

### Frontend
- **React** with modern hooks
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **Dark/Light mode** toggle
- **Responsive design** for all devices
- **Glassmorphism** UI effects
- **SEO optimized** with React Helmet

### Backend
- **Node.js** with Express.js
- **MongoDB** database
- **JWT authentication** for admin
- **RESTful API** architecture
- **Rate limiting** for security
- **Error handling** middleware

### Pages
- **Home Page** - Hero, About, Skills, Timeline
- **Projects Page** - Dynamic project showcase
- **Blogs Page** - Technical blog posts
- **Contact Page** - Message form
- **Admin Panel** - Dashboard with CRUD operations
